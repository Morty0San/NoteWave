NoteWave

Application web de prise de notes avec mémos vocaux intégrés.

- Stack technique
 **Frontend** : React.js (hooks, gestion d'état, JWT)
 **Backend** : Node.js / Express
 **Base de données** : MongoDB
 **Auth** : JWT (JSON Web Tokens)
 **Audio** : MediaRecorder API

- Fonctionnalités
   Authentification sécurisée (inscription / connexion)
   Création et édition de notes par dossier
   Enregistrement de mémos vocaux (max 2 min)
   Lecture et suppression des mémos par note
   Gestion de dossiers personnalisés
   Sauvegarde automatique

- Statut
   En cours de développement — finalisation août 2026
